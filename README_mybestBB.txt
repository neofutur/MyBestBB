MyBestBB 2.0.4, based on punbb 1.2.23 ( see http://punbb.org/ )

 MyBestBB includes many useful mods/plugins/additional features :
including :

-> Simple RSS 	: rss.php 		( see http://www.alexking.org/software/punbb/rss.zip )
-> Attachments 				( see http://www.frankh.se/?page_id=8 ) 
-> Movepost 	: admin can move posts 	( see http://www.punres.org/desc.php?pid=187 )
-> Movable Subforums			( see http://www.punres.org/desc.php?pid=209 )
-> Private messaging 			( see http://www.punbb.fr/forums/viewtopic.php?id=3218 )
-> Smilies 	: many easysmilies 	( see http://www.punres.org/desc.php?pid=37 )
-> Polls	: easypoll 		( see http://www.punres.org/desc.php?pid=60 )
-> Activity Barometer 			( see http://www.punres.org/desc.php?pid=142 )
-> Ajax_extern				( see http://forums.punbb.org/viewtopic.php?pid=64907#p64907 )
-> SMTPS ( gmail and other secure SMTP ) support
-> Active Topics: activetopics anywhere	( see http://www.punres.org/files.php?pid=252 )
-> Topic Rating	: topics can be rated	( missing mod url, contact us )
-> StickyTopics	: 			( missing mod url, contact us )
-> Reputation 	: reputation system	( missing mod url, contact us )
-> Signature 	: includes signture.png	( see http://www.punres.org/desc.php?pid=116 ) with forum stats ( needs libgd )
-> Board Logo : admin board logo option	( see http://www.punres.org/desc.php?pid=202 )
-> punrewrite : better SEO with rewrite	( see http://www.punres.org/desc.php?pid=139 )
-> Bulk Mail : Email Global Plus plugin	( see http://www.punbb.fr/mods/mod.php?id=22 )
-> Chatbox   	: new ajax chatbox 	( see http://forums.punbb.org/viewtopic.php?id=8174 )
-> Welcome Message to new user		( see http://www.punbb.fr/mods/mod.php?id=72 )
-> Bulk user management
-> Antibot features                     ( see http://www.network-technologies.org/Projects/Virtual_Brain_Online/article/spam_bot_registration_mod_punbb/ for more information )

 Additionaly :
-> default utf8 encoding ( you have to use iconv if migrationg from a non utf8 punbb )
-> Languages : English and French languages are currently supported ( please help add more languages )
   You can easyly add support for your language, contact us
-> strict XHTML compliance for each public page
-> better XHTML compliance for admin and plugins pages
-> added <link rel="alternate" type="application/rss+xml" title="Subscribe RSS FEED" href="/rss.php" /> to main.tpl
-> many improved themes
