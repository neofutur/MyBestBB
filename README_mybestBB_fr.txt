MyBestBB 2.0.2, se base sur punbb 1.2.22 ( voir http://punbb.org/ )

 MyBestBB inclue de nombreuses modifications utiles, qui ajoutent des fonctionnalites a votre punbb 1.2.*

sont inclues les fonctionnalites suivantes :

-> Simple RSS 	: rss.php 		( voir http://www.alexking.org/software/punbb/rss.zip )
-> Attachments 				( voir http://www.frankh.se/?page_id=8 ) 
-> Movepost 	: admin can move posts 	( voir http://www.punres.org/desc.php?pid=187 )
-> Movable subforums			( voir http://www.punres.org/desc.php?pid=209 )
-> Private messaging                    ( voir http://www.punbb.fr/forums/viewtopic.php?id=3218 )
-> Smilies 	: many easysmilies 	( voir http://www.punres.org/desc.php?pid=37 )
-> Polls	: easypoll 		( voir http://www.punres.org/desc.php?pid=60 )
-> Activity Barometer 			( voir http://www.punres.org/desc.php?pid=142 )
-> Ajax_extern				( voir http://forums.punbb.org/viewtopic.php?pid=64907#p64907 )
-> Support SMTPS ( gmail et autres SMTP securises )
-> Active Topics: activetopics anywhere	( voir http://www.punres.org/files.php?pid=252 )
-> Topic Rating	: topics can be rated	( missing mod url, contact us )
-> StickyTopics	: 			( missing mod url, contact us )
-> Reputation 	: reputation system	( missing mod url, contact us )
-> Signature 	: includes signture.png	( voir http://www.punres.org/desc.php?pid=116 ) with forum stats ( needs libgd )
-> Board Logo : admin board logo option	( voir http://www.punres.org/desc.php?pid=202 )
-> punrewrite : better SEO with rewrite	( voir http://www.punres.org/desc.php?pid=139 )
-> Bulk Mail : Email Global Plus plugin	( voir http://www.punbb.fr/mods/mod.php?id=22 )
-> Chatbox   	: new ajax chatbox 	( voir http://forums.punbb.org/viewtopic.php?id=8174 )
-> Welcome Message to new user          ( voir http://www.punbb.fr/mods/mod.php?id=72 )
-> Bulk user management
-> Antibot features                     ( see http://www.network-technologies.org/Projects/Virtual_Brain_Online/article/spam_bot_registration_mod_punbb/ for more information )

-> default utf8 encoding ( you have to use iconv if migrationg from a non utf8 punbb )
-> Languages : English and French languages are currently supported ( please help add more languages )
   You can easyly add support for your language, contact us
-> strict XHTML compliance for each public page
-> better XHTML compliance for admin and plugins pages
-> added <link rel="alternate" type="application/rss+xml" title="Subscribe RSS FEED" href="/rss.php" /> to main.tpl
-> many improved themes
-> dc:creator in rss
